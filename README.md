# ICMS — 교과서 읽기 AI 콘텐츠 자동 생성 등록 (프로토타입)

통합콘텐츠 관리자(ICMS)의 **국어 > 교과서 읽기** 콘텐츠를 AI로 자동 생성·등록하는 화면 흐름을
클릭으로 체험할 수 있는 **기획용 프로토타입**입니다. (모든 데이터는 더미입니다.)

## 화면 흐름

```
목록 ──[신규 등록]──▶ 등록(입력 전)
  │                      │
  │                      ├─[AI 콘텐츠 자동 생성]─▶ (처리중 팝업) ─▶ AI 출력 완료
  │                      └─[등록하기]─▶ 목록
  │
  └─[콘텐츠 코드 클릭]──▶ 상세 ──[수정]──▶ 수정(프리필) ──[수정하기]──▶ 상세
```

| 화면 | 설명 |
|------|------|
| **목록** | 교과서 읽기 콘텐츠 리스트 · 검색 필터 · 페이지네이션 |
| **등록 (CASE 1)** | 입력 전 기본 화면. AI 입력 영역만 노출, AI 출력은 비어 있음 |
| **등록 (CASE 2)** | `AI 콘텐츠 자동 생성` 클릭 시 처리중 로딩 팝업 |
| **등록 (CASE 3)** | 생성 완료 — 인트로·문단별 요약·퀴즈/정답이 채워지고 `등록하기` 활성 |
| **상세** | 등록된 콘텐츠 정보 확인 (콘텐츠/목차/메타/첨부/등록 정보) |
| **수정** | 등록 화면과 동일 폼에 데이터 프리필 (회차코드는 수정 불가) |

## 주요 인터랙션

- **AI 자동 생성**: 제목·본문 미입력 시 안내 토스트 → 입력 후 클릭하면 로딩 팝업 후 결과 채움
- **AI 출력 편집**: 인트로·문단·퀴즈 직접 수정 / `+ 문단 추가`, `+ 퀴즈 추가`
- **퀴즈 정답**: 보기 좌측 라디오로 정답 지정 · 보기 추가/삭제(최소 2개) · 정답 보기 삭제 시 첫 보기로 초기화
- **유효성**: 필수값 미입력 시 등록/수정 차단 안내

## 화면 바로가기 (딥링크)

URL 뒤에 해시를 붙이면 특정 화면으로 바로 진입합니다.

- `#list` — 목록
- `#register` — 등록(입력 전)
- `#detail` — 상세
- `#edit` — 수정(프리필)
